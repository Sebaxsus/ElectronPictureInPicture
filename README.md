# Aplicación en electron Picture In Picture.

## Descripción General

Aplicación de electron para reproducir contenido web en modo Picture In Picture cuando la pagina web no lo soporta o no tiene esa característica.

## Que hace?

1. Mostrar contenido en modo picture in picture.
1. Saltar anuncios (Usando un Identificador único del elemento).
1. Que el video ocupe toda la ventana.
1. Incluir los controles (Volumen, Saltar, Devolverse, Adelantar, Retroceder, Volver a un tamaño razonable para interactuar con la pag web).
